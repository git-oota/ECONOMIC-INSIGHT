お金の価値は
「金利」と「為替」で決まる

「100円」は明日も同じ価値でしょうか？
中学生のうちに知っておきたい、経済の最強ルール。

金利 ＝ お金の「レンタル料」

お金を銀行に預けることは、銀行に「貸す」ということです。そのお礼として受け取るのが利子、その割合が金利です。

金利が高い国

銀行が「もっと貸して！」と言っている状態。お金を預ける人が増えます。

金利が低い国

銀行にお金が余っている状態。預けてもあまり増えません。

インフレ ＝ 物価の上昇

物価が上がると、同じお金で買えるものが少なくなります。

去年: 100円

今年: 103円

「実質金利」を計算しよう

銀行の金利が上がっても、物価がそれ以上に上がったら損をします。これが実質金利のワナです。

日本の現状

日本は現在、物価の上昇に金利が追いついていないため、貯金をしているだけでは価値が目減りする特殊な状態です。

為替と円安・円高

為替（かわせ）とは、お金とお金の「交換レート」のこと。シーソーのように価値が動きます。

円安
