金利（きんり）＝ お金の「レンタル料」

お金を銀行に預けることは、銀行に「お金を貸す」ということです。
そのお礼として受け取るのが利子（りし）、その割合が金利です。

金利が高い国（例：アメリカ 5%）

100万円預けると、1年後に105万円になる！

→ お金を預けたい人が増える

金利が低い国（例：日本 0.25%）

100万円預けても、1年後に100万2500円...

→ 預けてもあまり増えない

インフレ ＝ モノの値段が上がること

「インフレ率3%」とは、「去年100円だったものが今年は103円になった」ということです。

去年: 100円

今年: 103円

インフレ率が 3% だと、お金の価値が相対的に下がります。

「実質金利」を計算してみよう

銀行の金利が上がっても、物価がそれ以上に上がったら、実は損をしているかもしれません。

名目金利 (銀行) − インフレ率 ＝ 実質金利

あなたの本当の利回り

-2.5%

お金の価値が1年でこれだけ減っています。

日本の「実質マイナス」の現状

日本は今、歴史的に見て珍しい状態にあります。金利を上げ始めていますが、物価の上昇に追いついていません。

銀行の金利: 低い
インフレ率: 高い (約3%)
実質金利: マイナス

為替（かわせ）と円安・円高

為替とは、国と国のお金の「交換レート」のことです。

円安 (えんやす)

1ドル ＝ 150円

円の価値が下がり、ドルの価値が上がった状態。

現在はこちらの傾向

円高 (えんだか)

1ドル ＝ 100円

円の価値が上がり、ドルの価値が下がった状態。

なぜ「円安」が進んだの？

最大の理由は、日本とアメリカの「金利の差」です。

日本

0.25%

お金が移動！

アメリカ

5.0%

「金利が高いドルで持っていたほうが得だ！」
世界中の投資家が円を売ってドルを買うため、円安になります。

私たちの生活への影響

困ること (円安＋インフレ)

  • エネルギー高
    ガソリンや電気が高くなる。
  • 食品の値上げ
    小麦粉や油、肉など輸入品が高い。
  • スマホやゲーム
    海外メーカーの製品が値上がりする。

良いこと (一部の人)

  • 輸出企業
    トヨタなど、海外で売る企業の利益が増える。
  • 観光業
    外国人観光客が「安く旅行できる」と日本に来る。

日本とアメリカはどう考えている？

アメリカ (FRB)

「インフレは経済の敵だ！金利を高くして、みんなが無理に買い物しないようにして、物価を落ち着かせよう。」

日本 (日銀)

「急に金利を上げると、企業の借金や住宅ローンが払えなくなる人が増えて不景気になるのが怖い。給料が上がるのを待ちたい。」

日本は「ゆるやかな金利上昇」を目指していますが、
アメリカとの差がある限り、円安圧力は続きます。
